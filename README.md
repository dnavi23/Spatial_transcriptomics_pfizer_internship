# TLS Signature Evaluation Pipeline

## 🎯 Overview

A comprehensive pipeline for evaluating Tertiary Lymphoid Structure (TLS) gene signatures on 10x Visium spatial transcriptomics data. This workflow integrates **BayesSpace spatial enhancement** with **SpaCET cell type deconvolution** into a unified Seurat object, enabling TLS detection at enhanced subspot resolution with cell type context.

### Key Innovation: BayesSpace-SpaCET Integration

**The Novel Contribution**: This pipeline creates a novel "bayespacet-seurat" object that seamlessly integrates:
- **Enhanced spatial resolution** from BayesSpace (subspots instead of spots)
- **Cell type proportions** from SpaCET deconvolution
- **Unified format** in Seurat for downstream analysis

**Technical Achievement**: Handles complex format conversions:
- Barcode format standardization across three object types
- Coordinate system transformations (tissue ↔ pixel space)
- Matrix format conversions (dense ↔ sparse)
- Dimension and barcode alignment

**Workflow Steps:**
1. **Data Preparation**: Load 10x Visium spatial transcriptomics data
2. **Spatial Enhancement**: Run BayesSpace to create subspot resolution
3. **Format Conversion**: Convert BayesSpace SCE to SpaCET-compatible format
4. **Cell Type Deconvolution**: Run SpaCET to get cell type proportions
5. **Integration**: Create unified bayespacet-seurat object
6. **Signature Evaluation**: Calculate and evaluate TLS signatures
7. **Performance Metrics**: Generate ROC and PR curves

## 🚀 Quick Start

### Prerequisites

```bash
# Install R packages
Rscript -e "install.packages(c('data.table', 'ggpubr', 'ggplot2', 'PRROC', 'pROC', 'ROCit', 'jsonlite', 'argparse', 'rmarkdown'))"
Rscript -e "BiocManager::install(c('BayesSpace', 'SpaCET', 'Seurat', 'SingleCellExperiment'))"

# Install Snakemake
pip install snakemake
```

### Option 1: Run with Snakemake (Recommended)

```bash
# 1. Download your Visium data to data/{sample}_outs/
# 2. Update config.yaml with your sample IDs and cancer type
# 3. Run pipeline
snakemake --cores 8
```

### Option 2: Run R Markdown Directly

```r
# Open TLS_signature_evaluation.Rmd
# Update paths in Section 3
# Run all chunks
```

## 🛠️ Key Functions

### `create_spacet_from_bayespace()`
Converts BayesSpace SCE to SpaCET format:
- Creates unified barcodes
- Transforms coordinates (tissue → pixel space)
- Converts matrix formats

### `create_bayespacet_seurat()`
Creates integrated Seurat object:
- Combines BayesSpace counts + SpaCET proportions
- Transforms coordinates (pixel → tissue space)
- Aligns all dimensions and barcodes

### `calculate_tls_signatures()`
Calculates module scores for all TLS signatures

### `evaluate_tls_signatures()`
Evaluates performance using ROC and PR curves
